# 천천히 톡톡

60대 이상 사용자가 혼자서도 따라갈 수 있도록 구성한 어깨 톡톡 연습 UI 프리뷰입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 구현된 흐름

- 불편한 어깨를 큰 글자 버튼으로 선택하고 반대편 팔을 자동 안내
- 휴대폰 거치와 팔 위치 맞추기를 먼저 안내
- 실제 브라우저 카메라 권한 요청, 미리보기, 실패 시 대체 경로
- 타점 찾기와 톡톡 수행 분리
- 3초 카운트다운, 횟수 중심 진행, 일시정지와 계속하기
- 단계가 끝난 뒤 사용자 선택으로 다음 타점 이동 또는 반복
- 브라우저 한국어 음성 안내, 박자음, 모바일 진동
- 큰 글씨 모드, 명확한 키보드 포커스, 동작 최소화 설정 대응

현재 자동 횟수 표시는 비전 모델 대신 `registerDetectedHit()`를 주기적으로 호출해
시뮬레이션합니다. 화면의 타점과 횟수는 UI 체험용 예시이며 의료 안내가 아닙니다.
